This website will be a basic fansite about Random Access Mayhem, an indie game that I'm a huge fan of. I plan to include articles on the playable characters/enemies, the lore, and the characters.

  • Main Page: An overview of RAM, including general information, an external link to the official Steam store page, as well as a section with some tips and tricks for beginners.
  • Host Information - RAM has 8 enemies, each of which has a unique moveset that the player can take control of. This page will be a short overview of each playable bot, with some basic information, upgrade recommendations, and trivia.
  • The World of RAM - This page will cover various characters and places, and how they relate to each other. I'm considering implimenting Javascript in order to create a "toggle spoilers" button at the top of the page.
  • RAM Secrets and Lore - This page will cover some of the more obscure or hidden aspects of RAM's worldbuilding and lore, as well as gameplay oddities or unique trivia. I could even repurpose the quote generator code in order to create a secret paragraph that has a small random chance to appear whenever the webpage is loaded.
